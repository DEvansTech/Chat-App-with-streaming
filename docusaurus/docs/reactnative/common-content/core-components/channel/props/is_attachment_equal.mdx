Custom function which accepts previous and next attachment and return boolean according to user needs.
This is useful to implement custom logic for memoization. This is helpful when using custom properties in attachment object.

| Type     |
| -------- |
| function |

| Parameter | Description                                |
| --------- | ------------------------------------------ |
| prevProps | previous message attachment to be compared |
| nextProps | next message attachment to be compared     |
