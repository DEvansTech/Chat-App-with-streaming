List of reactions that can be added to a message. Provide `id` and `icon` for every reaction that you want to support.
`id` is an identifier for reaction, which is used by the backend. `icon` is used to to render the UI for that reaction.

Please read [Custom Reaction section](/reactnative/guides/message-customization#message-with-custom-reactions) in Guides section.

| Type | Default |
| - | - |
