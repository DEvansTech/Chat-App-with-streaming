Use this prop to override following default UI components, which are part of [image gallery](../../ui-components/image-gallery).

- [ImageGalleryFooter](https://github.com/GetStream/stream-chat-react-native/blob/master/src/components/ImageGallery/components/ImageGalleryFooter.tsx)
- [ImageGrid](https://github.com/GetStream/stream-chat-react-native/blob/master/src/components/ImageGallery/components/ImageGrid.tsx)
- [ImageGridHandle](https://github.com/GetStream/stream-chat-react-native/blob/master/src/components/ImageGallery/components/ImageGridHandle.tsx)
- [ImageGalleryHeader](https://github.com/GetStream/stream-chat-react-native/blob/master/src/components/ImageGallery/components/ImageGalleryHeader.tsx)

```jsx
{
 footer: {
   ShareIcon: CustomShareIconComponent
 },
 grid: {
   avatarComponent: CustomAvatarComponent
 },
 gridHandle: {
   centerComponent: CustomCenterComponent
 },
 header: {
   CloseIcon: CustomCloseButtonComponent
 },
}
```
