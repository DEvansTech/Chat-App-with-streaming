`InputButtons` is a UI component to render container for `AttachButton` and `CommandsButton` on left side of input box, within `MessageInput` component.

You can replace this default UI component using the `InputButtons` prop on [Channel](../core-components/channel) component.

```tsx
<Channel InputButtons={CustomInputButtons} />
```
