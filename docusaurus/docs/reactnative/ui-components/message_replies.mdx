`MessageReplies` is a UI component to render thread reply count and avatars of users who replied to the message in thread. Avatars are rendered using [`MessageRepliesAvatar`](./message-replies-avatar) component.

You can replace this default UI component using the `MessageReplies` prop on [Channel](../core-components/channel) component.

```tsx
<Channel MessageReplies={CustomMessageReplies} />
```

## Props

:::note
This component uses default values for all the **non-required** props, from [`MessageContext`](../contexts/message-context)
:::

### <div class="label required">required</div> **noBorder**

If true, little curved line next to the reply count won't be rendered.

| Type |
| - |
| boolean |

### <div class="label required">required</div> **repliesCurveColor**

Color for the curved line next to the reply count.

| Type |
| - |
| string |
