title: ScrollToBottomButton
---

`ScrollToBottomButton` is a UI component to render a button, which when pressed scrolls the `MessageList` to the most recent message.

## Props

### onPress

`onPress` handler for the button.

| Type |
| - |
| `() => void` |

### showNotification

When false, button won't be rendered.

| Type |
| - |
| boolean |

### unreadCount

Unread count on current channel.

| Type |
| - |
| number |
