The `ChannelPreviewTitle` is the UI component for displaying the title for each channel on a list.

![](../assets/api-references/components/channel-list/visual_guide_1.png) <!-- Replace with propert screenshot -->

## UI Customizations

You can have a custom channel preview title component by setting the `PreviewTitle` prop on any of its parent UI components.

```jsx
import { ChannelPreviewMessenger } from 'stream-chat-react-native';

const CustomPreviewTitle = ({ displayName }) => {
    return (
        <Text>Title: {displayName}</Text>
    );
}

<ChannelPreviewMessenger PreviewTitle={CustomPreviewTitle} />
```
