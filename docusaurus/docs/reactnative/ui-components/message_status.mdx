`MessageStatus` is a UI component which renders the read recipts for message. If the message is being sent (`message.status === 'pending'`), then it will render pending state icon.

:::tip
You can disable read recipts at all by using [`readEventsEnabled`](../core-components/channel#readeventsenabled) prop on `Channel` component.
:::
