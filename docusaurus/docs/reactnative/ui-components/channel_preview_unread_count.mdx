The `ChannelPreviewUnreadCount` is the UI component for displaying the unread count badge for each channel on a list.

![](../assets/api-references/components/channel-list/visual_guide_1.png) <!-- Replace with propert screenshot -->

## UI Customizations

You can have a custom channel preview unread count component by setting the `PreviewUnreadCount` prop on any of its parent UI components.

```jsx
import { ChannelPreviewMessenger } from 'stream-chat-react-native';

const CustomPreviewUnreadCount = ({ unread, maxUnreadCount }) => {
    return (
        <View style={[someOtherStyle, { backgroundColor: 'blue' }]}>
            <Text numberOfLines={1}>
                {unread > maxUnreadCount ? `${maxUnreadCount}+` : unread}
            </Text>
        </View>
    );
}

<ChannelPreviewMessenger PreviewUnreadCount={CustomPreviewUnreadCount} />
```
