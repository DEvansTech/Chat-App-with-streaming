The `ChannelPreviewMessenger` is the UI component for channel previews. Renders a preview item for each channel in a list.

You can replace the channel preview UI component using the `Preview` property on [ChannelPreview](./channel-preview#preview)<!-- TODO: Should we document channel preview? --> or [ChannelList](../core-components/channellist#preview).

![](../assets/api-references/components/channel-list/visual_guide_1.png)

## UI Customizations

`ChannelPreviewMessenger` component makes it possible to customize any of the enclosed child components. For every child UI component of `ChannelPreviewMessenger`, you can provide your own custom implementation. You can reference the visuals in the first section, to check which prop to use for customizing.

```jsx
import { ChannelPreviewMessenger } from 'stream-chat-react-native';

const CustomPreviewTitle = ({ channel }) => (
  <Text>{channel.data.customProperty} - {channel.data.name}</Text>
)

<ChannelPreviewMessenger PreviewTitle={CustomPreviewTitle} />
```
